# 청년 AI·Big Data 아카데미 온라인 기초과정(MOOC)

- POSTECH과 POSCO가 공동으로 제공하는 『청년 AI · Big Data 아카데미』


## 교육 내용
- 온라인 기초과정
  - 1. AI·Big Data 활용을 위한 기본원리와 프로그래밍 학습
  - 2. Big Data 의 원리와 활용기법 학습
  - 3. AI의 원리와 활용기법 학습
  - *특강: 블록체인, AI, IoT, 컴퓨터 비전, 데이터사이언스
